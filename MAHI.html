Machine Availability Human Index (MAHI) Dashboard

This project features the Machine Availability Human Index (MAHI) Dashboard, which is designed to monitor and visualize machine performance in a manufacturing environment. The dashboard provides insights into machine uptime, downtime, and efficiency, enabling operational teams to make data-driven decisions for continuous improvement.

Overview

The MAHI dashboard helps track machine availability and its impact on production performance. It highlights key metrics such as:

  • Machine Uptime: The total time a machine is available for use.
  • Downtime Analysis: Breakdown of reasons for downtime, including maintenance and failure.
  • Efficiency Metrics: Insights into machine performance over time.
The dashboard is built using Tableau, featuring interactive filters and visualizations such as bar charts, KPIs, and line graphs to provide actionable insights into machine performance.

Key Features

  • Machine Uptime & Downtime Monitoring: Real-time monitoring of machine availability and downtime reasons.
  • Root Cause Analysis: Identify trends in machine downtime and diagnose root causes for failure.
  • Efficiency Metrics: Visual representation of machine efficiency, highlighting performance over time.
  • Interactive Filters: Use filters for date ranges, machine types, and departments to drill down into the data.

Usage

The dashboard is designed to provide managers, engineers, and other stakeholders with a clear overview of the operational health of machines within a production facility. It is ideal for:

  • Manufacturing Performance Monitoring
  • Maintenance Optimization
  • Production Planning

Skills & Technologies Used

  • Tableau: Used to build interactive dashboards and visualize machine performance data.
  • SQL: Used for querying production and machine performance data from databases.
  • Excel: Used for data cleaning and preparation before importing to Tableau.
  • Data Analysis: Utilized to generate insights into machine efficiency and downtime.
